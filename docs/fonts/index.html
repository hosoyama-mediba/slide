インストール方法
Homebrew 
$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
brew-cask 
$ brew install caskroom/cask/brew-cask
フォント一覧 
$ brew cask search /font/
欲しいフォントがあるか探す 
$ brew cask search /font/ | grep ricty
インストール 
$ brew cask install font-ricty-diminished

Sublime Textの設定

{
    "font_face": "Ricty Diminished Discord",
    "font_size": 16,
}
GitHubのレビュー時にフォントを指定するにはChrome拡張を使ってCSSを上書きする 
.blob-code-inner {
    font-family: "Ricty Diminished Discord", Ricty, Consolas, "Liberation Mono", Menlo, Courier, monospace !important;
    font-size: 14px !important;
}
